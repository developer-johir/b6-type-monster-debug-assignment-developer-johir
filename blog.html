Question-1: What is the difference between LocalStorage and SessionStorage?

Answer:- Session storage is similar to local storage; The difference is that while data in local storage does not expire, data in session storage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session is created and assigned to that particular tab.

Question-2: What is the difference between global scope and block scope?

Answer:- In a browser environment, the global scope is controlled by the window object while in Node. js, it's controlled by the global object. Block scopes are what you get when you use if statements, for statements, and the like.

Question-3: How does the event loop work in JavaScript?

Answer:- JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events, and executing queued sub-tasks. This model is quite different from models in other languages like C and Java.

Question-4: Why is undefined found in Javascript?

Answer:- A variable that has not been assigned a value is of type undefined. A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned.
